# Neovim Configuration

1. Run the script 'setup.sh'

2. Run interactively:

```bash
pyenv activate neovim2
pip install neovim
pip install jedi
pyenv deactivate

pyenv activate neovim3
pip install neovim
pip install jedi
pyenv deactivate

pyenv virtualenvwrapper_lazy
```

3. Select powerline font in the terminal profile
